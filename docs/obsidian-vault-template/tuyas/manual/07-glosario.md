# 07 — Glosario UFLO + PPS

> Vocabulario propio. Hermes lo usa para entender mensajes y para hablar como vos.

## Términos académicos UFLO

| Término                            | Significado                                                                                      |
| ---------------------------------- | ------------------------------------------------------------------------------------------------ |
| **PPS**                            | Práctica Profesional Supervisada                                                                 |
| **Cohorte**                        | Tanda de alumnos que arranca una PPS en una fecha                                                |
| **Acreditación**                   | Reconocimiento formal de horas de PPS cumplidas                                                  |
| **Convenio marco**                 | Acuerdo general UFLO ↔ Institución, válido por años                                              |
| **Convenio específico**            | Acuerdo por cohorte o por intervención puntual                                                   |
| **Tutor institucional**            | Profesional de la institución que supervisa al alumno                                            |
| **Docente referente**              | Docente UFLO que supervisa la PPS desde la facultad                                              |
| **Sede BA**                        | Sede Buenos Aires (Almagro / Recoleta)                                                           |
| **Sede Patagonia**                 | Sedes regionales (Cipolletti, Roca, Neuquén, etc.)                                               |
| **Anexo**                          | Sedes menores asociadas a Patagonia (San Miguel, Zapala, San Martín de los Andes, Plaza Huincul) |
| **Orientación**                    | Eje temático de la PPS (Clínica, Educacional, Laboral, Comunitaria)                              |
| **Cupo**                           | Posición disponible para un alumno en un lanzamiento                                             |
| **Lanzamiento** / **convocatoria** | La instancia de una PPS abierta a inscripciones                                                  |
| **Inscripto**                      | Alumno que se anotó pero todavía no fue seleccionado                                             |
| **Seleccionado**                   | Alumno que pasó el cut, va a hacer la PPS                                                        |
| **Seguro**                         | Cobertura estudiantil obligatoria antes del inicio                                               |

## Falsos amigos

Cosas que parecen iguales pero no son:

- **"PPS"** en UFLO = la materia. En el ámbito empresarial puede significar otra cosa, ignorar.
- **"Convenio"** sin contexto → casi siempre se refiere al institucional UFLO ↔ Institución.
- **"Cupo"** sin contexto → vacante en un lanzamiento, no en una materia.
